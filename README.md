# Toyota Gazoo Racing Argentina - Fan Page 🏎️🏁

Este es mi proyecto final de **Desarrollo Web**. Es una landing page diseñada para los fanáticos de la división deportiva de Toyota, enfocada en la velocidad, el rendimiento y el ADN de Gazoo Racing.

## 🚀 Tecnologías Aplicadas
* **HTML5**: Estructura semántica para una mejor navegación y SEO.
* **SASS**: Arquitectura de estilos modular y organizada.
* **Bootstrap 5**: Maquetación responsive, uso de Grid, Navbar con dropdown y utilidades.
* **Git & GitHub**: Control de versiones y despliegue del sitio.

## 🛠️ Características del Proyecto
* **Hero Section**: Encabezado de alto impacto con video de fondo (`autoplay/muted`).
* **Navegación Dinámica**: Menú desplegable y responsive (hamburguesa en móviles).
* **Galería GR**: Sección de modelos con tarjetas visuales.
* **Diseño Dark Mode**: Estética alineada a la identidad visual de Toyota Gazoo Racing.
* **Formularios y Secciones**: 5 secciones navegables cumpliendo con los requisitos del curso.
