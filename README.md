# Toyota Gazoo Racing - Fan Page 🏎️

Proyecto final para el curso de Desarrollo Web. Una página dedicada a la pasión por el automovilismo de Toyota.

## 🚀 Tecnologías utilizadas
* **HTML5**: Estructura semántica y SEO.
* **SASS**: Preprocesador de CSS para una arquitectura de estilos escalable.
* **Bootstrap 5**: Framework para diseño responsive y componentes dinámicos.
* **Git/GitHub**: Control de versiones y despliegue.

## 🛠️ Características
* Menú desplegable responsive.
* Hero section con video de fondo.
* Galería de modelos interactiva.
* Optimización para dispositivos móviles.
